## Limit: 2 Options

### Option A: Install Git & Run Script (Recommended)
1. **Download Git**: [https://git-scm.com/downloads](https://git-scm.com/downloads)
2. **Install**: Run the installer (click "Next" through defaults).
3. **Run Script**: Double-click the `push_code.bat` file I created in this folder.
   - It will automatically run all commands for you.
   - It connects to: `https://github.com/veddhanani4/Profitlens-Ai.git`

### Option B: Drag & Drop (Manual)
1. Open your repository: [https://github.com/veddhanani4/Profitlens-Ai](https://github.com/veddhanani4/Profitlens-Ai)
2. Click **Add file** ‚Üí **Upload files**.
3. Select all files in this folder (`Ctrl+A`) (excluding `node_modules` if present).
4. Drag them into the browser window.
5. Wait for upload to finish.
6. Type "Initial commit" in the commit message box.
7. Click **Commit changes**.
